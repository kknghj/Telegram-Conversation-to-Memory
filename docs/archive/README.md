# Documentation Archive

이 폴더에는 구현이 끝났거나 현행 기준으로 대체된 계획, 결정, 운영 작업, 사건 기록을 보관합니다.

archive 문서는 역사적 근거일 뿐 현재 작업 지침이 아닙니다. 새 작업은 다음 현행 문서에서 시작합니다.

- `../vision.md`: 제품 철학과 장기 비전
- `../future_roadmap.md`: 구현 진행 상태와 다음 구현 목표
- `../validation_plan.md`: 검증 진행 상태와 다음 검증 행동
- `../architecture.md`: 현재 런타임 구조

## 분류

- `plans/`: 구현 완료 또는 대체된 설계·변경 계획
- `decisions/`: 특정 기능 도입 당시의 결정 기록
- `operations/`: 완료된 일회성 데이터·운영 작업
- `incidents/`: 장애, 회귀, 복구 과정 기록

archive 문서를 다시 현행화할 때는 그대로 이동하지 말고, 현재 코드와 방향 문서를 재검증한 뒤 필요한 내용만 현행 문서에 반영합니다.
